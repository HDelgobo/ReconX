# ReconX
RECONX - Ferramenta de OSINT e Reconhecimento para pentest e cibersegurança. Automatiza consultas WHOIS, DNS, headers HTTP, scans Nmap (TCP/UDP/Stealth), detecção de WAF e geração de relatórios. Versões para Kali Linux e Termux (Android). Ideal para CTFs, laboratórios e testes autorizados.
